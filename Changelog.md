# Changelog

All notable changes to this project will be documented in this file.

This project was developed as a continuous implementation effort.
Formal semantic versioning begins from v0.1.0.

---

## [0.1.0] - 2026-02-14

### Added

- Initial implementation of a distributed network monitoring system.
- Python-based load balancer service.
- Support for multiple load balancing strategies.
- Health check mechanism for backend servers.
- Metrics collection for monitoring server performance.
- Frontend dashboard for visualizing system status.
- REST API for load balancer control and statistics.
- Docker Compose setup for multi-service deployment.
- Basic project documentation (README).
